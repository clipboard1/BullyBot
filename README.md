# BullyBot - Telegram bot based on aiogram
## What it can do ?
1. Bot learns from chat messages
2. You can use Chat-GPT thgrough the bot
3. Bot can send sticker and images that you send it
4. Bot can generate demotivators
## Installation
- `git clone https://github.com/clipboard1/BullyBot`
- `pip install -r requirements.txt`
- Enter your GPT API and Telegram API tokens
## Usage
- Launch via `python bot.py`
